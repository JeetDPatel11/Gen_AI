# [Gen_AI_Project]

An interactive, locally-hosted generative AI chatbot built to experiment with open-source Large Language Models (LLMs). This project leverages **LangChain** for orchestration, **Ollama** for local model serving, and **Streamlit** for a clean, user-friendly web interface. 

It also integrates **LangSmith** for full-stack LLM observability, allowing you to trace, evaluate, and debug prompt chains under the hood.

## 🚀 Features
* **Local LLM Execution:** Runs models locally using Ollama, ensuring data privacy and zero API inference costs.
* **Conversational Interface:** An intuitive web UI built with Streamlit.
* **Prompt Engineering:** Utilizes LangChain's `ChatPromptTemplate` for structured AI interactions.
* **Observability:** Built-in LangSmith tracing to monitor token usage, latency, and model outputs.

## 🛠️ Tech Stack
* **Language:** Python
* **Framework:** Streamlit
* **AI/Orchestration:** LangChain, Ollama
* **Observability:** LangSmith
